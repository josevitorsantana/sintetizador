Modulator

Vamos continuar usando o modulator para entender os blocos que compõem um sintetizador. O Modulator tem 6 painéis que serão discutidos a seguir.

Se você prefere um tutorial em vídeo, Thiago, um dos autores deste material, gravou um passo a passo de como configurar e usar o Modulator.

Audio Node Graph

O Audio Node Graph é o painel onde ocorre a construção do sintetizador, podendo ser colocados e ligados diferentes blocos para diferentes efeitos no som. Como mostrado anteriomente, a ligação dos blocos é feita com a tecla shift ao clicar em um bloco e em outro em seguida. Para desfazer a ligação, apenas repita o processo com blocos já ligados.

Palette

No painel Palette se encontram os blocos a serem colocados no Audio Node Graph. Basta clicar em um bloco e ele será adicionado ao Audio Node Graph.

Node Parameters

O painel Node Parameters serve para modificar o funcionamento de um bloco selecionado no Audio Node Graph, alterando seus parâmetros.

Presets

O painel Presets serve para salvar e reutilizar modificações feitas pelo usuário, ele também possui opções padrão já salvas como exemplos.

Você pode baixar os presets que fizemos na página anterior: Preset 01 - Só oscilador e Preset 02 - Oscilador + Gain

Piano Keyboard

É o teclado do sintetizador, utilizado para tocar as notas com as modificações estabelecidas. Também é possível mudar as oitavas, o portamento e o arpejo do instrumento nesse módulo. Você pode usar o mouse, as teclas do teclado de seu computador ou um piano digital/teclado musical conectado via MIDI.

Graph

O módulo Graph é um módulo exclusivo para visualização de onda, onde você pode ver uma representação em frequência (o espectro) e a forma de onda gerada quando tocar no teclado do piano.
